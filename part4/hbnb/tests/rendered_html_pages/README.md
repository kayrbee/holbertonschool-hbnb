# Rendered HTML & Validation Artifacts

This folder contains all assets related to validating our final frontend implementation.

## Rendered HTML Copies

Fully rendered versions of our final HTML pages, used for running W3C validation outside the Flask/Jinja environment.

## Rendering Tools

Copies of the Python scripts used to generate the rendered HTML output.

## W3C Validation Results

Saved results from running each page through the W3C HTML validator, including notes on any warnings or fixes applied.
